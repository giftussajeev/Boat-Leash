**Boat Leash for Minecraft Java 1.16.5**

![readmeimage](https://github.com/user-attachments/assets/7d10be42-ee38-4b85-a091-94dcd5f0a5a1)
This mod brings the modern Minecraft feature that lets players leash boats. In newer versions of the game, you can tie a leash to a boat just like you do with mobs. This keeps the boat close and under control during travel or builds.

The mod adds key details to make it work well. It supports leash strength, so the link holds firm even with tugs or pulls. Range gets tuned too—you set how far the boat can drift before it snaps back.
Picture boating across a vast ocean. Without this, boats float away fast. Now, leash it to your lead animal or player. It stays put, cuts frustration, and speeds up group trips.
Other tweaks cover edge cases. Leash breaks at set limits. Strength scales with mob types or items used. All this matches modern play without big changes to old saves. Players gain smooth water adventures right away.



Requires
Fabric Minecraft 1.16.5
Dependencies Fabric-api, Cloth-api, Mod Menu (optional)
